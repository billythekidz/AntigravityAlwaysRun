# Antigravity Always Run

Auto-accept / auto-run button clicker for **Antigravity IDE**. Scans the main document, iframes, and webviews every 3 seconds for buttons labeled *Accept*, *Run*, *Approve*, or *Confirm* — and clicks them automatically.

## Features

- 🔍 Deep scan across iframes and webview elements
- 🖱️ Configurable auto-click with optional scroll-into-view
- 📌 Scroll inspector — hover + press <kbd>Space</kbd> to capture a scrollable element
- 🎯 Draggable floating UI with collapse/expand
- 🛑 One-click stop button

## Usage

Paste the contents of [`agy-auto-accept.js`](agy-auto-accept.js) into your browser DevTools console, or inject it via a userscript manager / extension.
